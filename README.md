# android-util

Utility classes for Android

## Available Utility Classes

* `AppLog`
* `AppUtil`
* `CameraUtil`
* `FileUtil`
* `NetUtil`
* `StreamUtil`
* `StringUtil`
* `ToastMaster`
* `UIUtil`
